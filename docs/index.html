üöÄ Quick Start

Prerequisites

  • RTL-SDR dongle (RTL2832U-based)
  • VHF antenna (137MHz for METEOR-M, 145MHz for ISS)
  • Docker and Docker Compose v2
  • Raspberry Pi 4/5 or Linux machine

Installation

git clone https://github.com/milesburton/night-watch.git cd night-watch cp .env.example .env vi .env # Set your coordinates docker compose -f docker/compose.yaml up -d

Access dashboard at http://localhost:8002
